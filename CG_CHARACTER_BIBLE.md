# CG Character Bible

Use this file before generating or accepting any new character CG. Clothes, props, weather, and lighting can change to fit the scene. Face structure, hair silhouette, body type, age read, and core expression language must stay consistent with the `images/simulator/characters_v2/` portrait.

## Global CG Rules

- Keep main character faces sharp and readable. No motion blur, fog blur, heavy depth-of-field blur, or cropped-off faces for the named character.
- Do not change a character's apparent age, ethnicity, body type, height impression, or facial structure.
- Use the portrait as the identity lock and the story scene as the clothing/context layer.
- Avoid readable UI text, brand logos, QR codes, or signs unless the scene specifically needs them.
- Save accepted project CGs under `images/simulator/cg/` and only wire them into story/gallery after visual review.
